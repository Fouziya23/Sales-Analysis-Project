# Sales Analysis — SQL · Python · Power BI

End-to-end analysis of 2,823 sales orders across 19 countries using SQL, Python and Power BI.

---

## Tools Used
- **SQL (MySQL)** — data loading, querying, aggregations, window functions
- **Python** — EDA using Pandas, Matplotlib, Seaborn
- **Power BI** — interactive dashboard

---

## Dataset
- 2,823 rows · 25 columns
- Fields: Order details, Product Line, Customer, Country, Territory, Deal Size, Date

---

## Key Business Questions Answered

| Question | Finding |
|----------|---------|
| Which country drives most revenue? | USA |
| Top product line? | Classic Cars |
| Best performing month? | November |
| Top customer? | Euro Shopping Channel |
| Most common deal size? | Medium (60.68%) |
| Total revenue? | 1.79M |

---

## SQL Highlights
- Revenue by country, product line, customer
- Best month per year using GROUP BY
- Top 5 products using RANK() window function

- Data loading and cleaning with Pandas
- Revenue trends by country, product line, month, year
